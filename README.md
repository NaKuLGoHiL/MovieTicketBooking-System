# Movie Ticket Booking System

A **web-based Movie Ticket Booking System** built using **Python**, **Flask**, **HTML**, **CSS**, and **MySQL**.  
This project allows users to view movies, select show times and seats, and book tickets through a simple web interface. It demonstrates **backend logic, database integration, and frontend design**.

---

## Features

- View available movies and show times  
- Select seats from a seat layout  
- Book movie tickets and confirm booking  
- Store bookings in **MySQL database**  
- Simple, clean **HTML/CSS frontend**  
- Admin panel to add/remove movies  
- User login and registration  
- Booking history and ticket printing  

---

## Technologies Used

- **Python** – Backend logic  
- **Flask** – Web framework  
- **MySQL** – Database for movies, seats, and bookings  
- **HTML & CSS** – Frontend and UI design
